## 개요

테마 **산출(Export)·버전 관리·접근성 자동화**를 강화함. 디자인 편집 옵션을 추가하고, 미세조정 반영 범위를 정교화함.

- **산출물:** `WEMB Studio/index.html`

## 주요 작업 내역

| 구분          | 작업 내용                                                                                        | 상태 |
| ------------- | ------------------------------------------------------------------------------------------------ | ---- |
| 디자인 편집   | 모서리 둥글기(border-radius) 조정을 '대시보드 편집 > 디자인 편집' 토글로 추가                    | 완료 |
| 반영 범위     | 미세조정 시 좌측 메뉴 색은 고정, 우측 콘텐츠에만 반영                                            | 완료 |
| 테마 관리     | 스냅샷 명명 저장, A/B 2분할 비교 뷰(썸네일·팔레트 포함), 변경 히스토리/되돌리기                  | 완료 |
| Export        | CSS 변수·JSON 토큰·Figma 변수·Tailwind config 출력+클릭 복사, PNG/SVG 캡처, 상태 직렬화 공유 URL | 완료 |
| 접근성 자동화 | 역할쌍(배경↔텍스트, 배경↔포인트) WCAG 대비비 자동 표, 미달 시 통과 수준 자동 보정                | 완료 |
| 컴포넌트      | 상태 갤러리(컴포넌트 시트) 추가                                                                  | 완료 |

## 핵심 기능 요약

- **산출물 확보:** 실무에 바로 쓰는 토큰(CSS/JSON/Figma/Tailwind)·이미지·공유 URL
- **버전 비교:** 두 테마를 나란히 비교하고 이력으로 되돌리기
- **접근성 자동 보정:** 대비 미달 색을 통과 수준으로 자동 교정

## 기술 구성

- **구조:** 단일 HTML + CSS 변수 토큰 + Vanilla JS
- **Export:** 상태 직렬화(encode/decode) 기반 공유 URL, 캔버스 캡처로 PNG
- **접근성:** 역할쌍 대비비 계산 → 미달 명도 보정

## 이슈 & 해결

- A/B 비교 콘텐츠가 빈약하고 잘림 → 비교 콘텐츠를 채우고 하단 썸네일·색상 동시 표시
- 이미지 뒤 격자 배경이 산만함 → 제거
- 통과 불가능한 색이 통과로 표기되는 정황 → 검증 로직 점검
