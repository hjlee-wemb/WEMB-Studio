## 개요

사용성·인터랙션 집중 개선일. 로고/헤더/이미지 테마추출의 **비개발자 UX**를 다듬고, 콘텐츠 직접 편집 기능과 실시간 요소(시계·날씨·랜덤 팔레트)를 추가함. 고객사 운영자·PM 관점의 UX 평가를 병행하고 비교본을 제작함.

- **산출물:** `WEMB Studio/index.html` (+ 비교본 `index02.html`)

## 핵심 기능 요약

- **친절한 테마 저작:** 색 지식 없는 사용자를 위한 아이콘형 업로드·툴팁·단계 정리
- **콘텐츠 직접 편집:** 카드 배치·텍스트를 사용자가 자유롭게 수정
- **실시간 위젯:** 시계·날씨·랜덤 팔레트로 관제 화면 현실감 강화

## 기술 구성

- **구조:** 단일 HTML + CSS 변수 토큰 + Vanilla JS
- **인터랙션:** hover/active 상태를 포인트 컬러 변수와 연동해 테마 전환 시 자동 일관성 유지
- **상태:** 편집/배치 상태 관리

## 이슈 & 해결

- 편집 모드에서 hover가 이중 적용 → 텍스트 입력 커서 신호만 남기고 색상 강조 제거
- 효과·그림자 과다로 시각 피로 피드백 → 버튼 그림자 제거, 강약을 농도로 표현
- 헤더 하단 라인 hover가 직관성 낮음 → hover/active를 '둥근 배경 틴트' 언어로 통일(농도·링으로 위계)
