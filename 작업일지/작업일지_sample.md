## 개요

WEMB Studio(컬러 테마 스튜디오) 화면에서 **브랜드 로고를 실제 로고 파일로 교체**하고, **라이트/다크 테마에 따라 로고 색상이 자동 전환**되도록 처리함. 아울러 "주요 업무 현황" 테이블 헤더 밴드 셀들이 서로 붙어 있던 문제를 **셀 간 4px 간격**으로 분리함.

- **산출물:** `WEMB Studio/index.html` (단일 self-contained 파일, 외부 의존성 없음)
- **로고 리소스:** `WEMB Studio/logo.svg` (WEMB 워드마크, 컬러 원본)
- **미리보기:** 로컬 정적 서버(launch: `wemb-studio` / 대체: `svg-preview`)

## 주요 작업 내역

| 구분        | 작업 내용                                                                                            | 상태 |
| ----------- | ---------------------------------------------------------------------------------------------------- | ---- |
| 로고 교체   | 텍스트로 표시되던 로고 2곳(사이드바 "W" 배지, 헤더 "WEMB" 텍스트)을 `logo.svg` 이미지로 교체         | 완료 |
| 테마 대응   | 라이트 = 원본 컬러 로고 / 다크 = 전체 흰색 로고, 테마 버튼에 따라 즉시 전환                          | 완료 |
| 상태 연동   | 테마 적용 함수(`apply()`)에서 현재 모드를 `<html data-mode>`에 반영하여 CSS가 테마를 인식하도록 처리 | 완료 |
| 테이블 간격 | `.bandh td`(헤더 밴드 알약 셀) 사이가 붙어 있던 문제를 4px 간격으로 분리                             | 완료 |

## 상세 작업 내용

### 1. 로고 교체 및 테마별 색상 대응

- **요구사항:** 로고를 `logo.svg`로 교체 / 라이트 = 컬러(파랑+끝글자 초록), 다크 = 전체 흰색
- **적용:** 두 위치의 텍스트를 이미지로 교체
  ```html
  <span class="logo"><img class="wlogo" src="logo.svg" alt="WEMB" /></span>
  ```
- **다크 테마 색상 처리:** CSS 필터로 로고 전체를 흰색으로 변환
  ```css
  html[data-mode='dark'] .wlogo {
    filter: brightness(0) invert(1); /* 로고 전체를 흰색으로 */
  }
  ```
- **테마 상태 연동:** `apply()`에 아래 한 줄 추가 → Dark/Light 전환 시 로고 색상 즉시 반영
  ```javascript
  root.setAttribute('data-mode', state.mode);
  ```

### 2. `.bandh` 테이블 셀 간격 조정

- **요구사항:** `.bandh td` 알약 셀들을 서로 4px씩 띄우기
- **원인:** 테이블이 `border-collapse: collapse` 상태라 `margin`/`gap` 직접 적용 불가, 알약 배경이 맞닿음
- **적용:** `background-clip: padding-box` + 좌우 투명 테두리로 실제 간격 생성
  ```css
  .bandh td {
    background: var(--bg-accent);
    background-clip: padding-box; /* 배경을 테두리 안쪽까지만 칠함 */
    border-left: 2px solid transparent; /* 좌우 2px 투명 여백 */
    border-right: 2px solid transparent;
    /* ... 기존 스타일 유지 ... */
  }
  ```
  → 인접 셀이 각각 2px씩 안쪽으로 들어가 알약 배경 사이가 **2px + 2px = 4px** 벌어짐

## 검증 결과

| 검증 항목        | 방법              | 결과                 |
| ---------------- | ----------------- | -------------------- |
| 라이트 테마 로고 | 브라우저 미리보기 | 컬러(파랑) 정상 표시 |
| 다크 테마 로고   | 브라우저 미리보기 | 전체 흰색 정상 표시  |
| 알약 셀 간격     | DOM 픽셀 실측     | 배경 사이 4px 확인   |

## 이슈 & 해결

- 경로에 공백이 있어(`WEMB Studio`) 전용 미리보기 서버 기동 실패 → 홈 디렉터리를 서빙하는 기존 `svg-preview` 서버로 서브경로 접근하여 확인
- 미리보기 스크린샷 도구 응답 지연 → 간격은 스크린샷 대신 DOM에서 픽셀값을 직접 측정하여 검증(간격 검증에는 실측값이 더 정확)

## 참고 사항

- 간격을 더/덜 주려면 `.bandh td`의 좌우 `2px` 값 조절(예: 좌우 `4px` → 8px 간격)
- `logo.svg`와 `index.html`이 동일 폴더에 있어 `src="logo.svg"` 상대경로로 동작
