## 개요

WEMB Smart Factory 종합현황 대시보드를 **웹(단일 HTML)** 으로 재현하고, 시드·트렌드 컬러 기반으로 **테마를 즉시 전환**하는 컬러 스튜디오의 기반을 구축함. Figma `system` 원본 디자인을 코드로 정밀 재현하고, 컬러 변수(디자인 토큰) 체계를 확립함. 도구 명칭을 **WEMB Studio**로 확정.

- **산출물:** `WEMB Studio/index.html` (단일 self-contained 파일)
- **소스 관리:** Git 저장소 연동

## 핵심 기능 요약

- **즉시 리테마:** 색 하나만 바꾸면 KPI·차트·도넛·테이블까지 전체 색이 일괄 전환
- **정밀 클론:** 실제 Figma 디자인의 차트 종류·축·범례·수치까지 일치
- **반응형·애니메이션:** 화면 폭 유동 그리드 + 진입/실시간 애니메이션

## 기술 구성

- **구조:** 단일 HTML + CSS 변수(디자인 토큰) + Vanilla JS (프레임워크·차트 라이브러리 미사용)
- **차트:** SVG / CSS(conic-gradient·그라데이션)로 직접 구현 → 색상 전부 CSS 변수 연동
- **테마 파이프라인:** `시드 → 팔레트 생성(genPalette) → CSS 변수 세팅`

## 이슈 & 해결

- 좁은 컬럼에서 패널 탭이 줄바꿈되어 제목 위 여백 발생 → 탭 한 줄 고정으로 정렬 바로잡음
- 100vh 화면 맞춤 시 일부 밀집 패널 스크롤 → 제목/본문 컨테이너 분리로 겹침 방지
- Figma 원본과 차트 디테일(범례·축·카드배경) 불일치 → 원본 재확인 후 재현
