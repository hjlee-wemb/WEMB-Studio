## 개요

컬러 컨트롤 구조를 그룹형으로 재편하고, **형상관리·공개 배포**(GitHub/Pages)를 구축함. 접근성(WCAG AAA)과 실시간 KPI 표현을 강화함.

- **산출물:** `WEMB Studio/index.html`
- **배포:** GitHub 저장소 2곳 등록 + GitHub Pages 공개

## 주요 작업 내역

| 구분          | 작업 내용                                                                        | 상태 |
| ------------- | -------------------------------------------------------------------------------- | ---- |
| 컨트롤 구조화 | 색상 변경 메뉴를 그룹별 접이식 드롭다운으로 재편, 활성 테마 외 소스 disable 처리 | 완료 |
| 배색 방식     | 배색 방식(harmony)을 기준 색 하위 메뉴로 편입                                    | 완료 |
| 추천 테마     | 다크/라이트 탭으로 구분 추천, 활성 색상과 UI 정합                                | 완료 |
| 팔레트 편집   | 색상 칩 피커·텍스트 직접 지정·복사 아이콘·초기화 버튼, 툴팁, 개별 항목 접힘 기본 | 완료 |
| 접근성        | WCAG **AAA 통과 색만** 콘텐츠 색상으로 사용하도록 필터링                         | 완료 |
| 실시간 KPI    | 기존값→목표값 카운트업, 3초 주기 갱신 + 상승 애니메이션                          | 완료 |
| 형상관리      | GitHub 저장소 2곳 등록, SSH 키 설정·인증 테스트, GitHub Pages 공개               | 완료 |
| 디테일        | 시작 모드(라이트/다크) 조정, 맨 왼쪽 색 가시성 개선, 타 기관 지연 뱃지 색상 구분 | 완료 |

## 핵심 기능 요약

- **정돈된 컨트롤:** 그룹 접이식 + 활성 소스만 노출로 조작 혼선 최소화
- **접근성 보장:** AAA 미달 색은 콘텐츠에 사용하지 않도록 원천 차단
- **공개 배포:** GitHub Pages로 실사용 링크 확보

## 기술 구성

- **구조:** 단일 HTML + CSS 변수 토큰 + Vanilla JS
- **접근성:** 상대휘도 기반 대비비 계산 → AAA 필터
- **배포:** 정적 호스팅(GitHub Pages), 다중 원격 push

## 이슈 & 해결

- SSH 키 이름 중복 우려 → 설정 확인 후 정상 인증 검증
- KPI 수치가 급변해 부자연스러움 → 카운트업 방식 + 3초 주기로 변동 제어
- 추천 테마에서 이전 선택이 계속 활성 표시 → 활성 색상 기준으로 상태 동기화
